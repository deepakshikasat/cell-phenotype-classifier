Cell Phenotype Classifier

Bench to Bytes portfolio analysis report. Uses only public or synthetic data.

Hypothesis

A transfer-learning image classifier can distinguish broad cell phenotypes and use GradCAM-style inspection to check whether image regions are biologically meaningful.

Data

Public-data-ready project structure for Broad Bioimage Benchmark Collection datasets, with a synthetic manifest and demo metrics included for reproducibility.

Process

  1. Created an image manifest with phenotype and split labels.
  2. Stored demo classification metrics and a confusion matrix.
  3. Outlined a ResNet18 transfer-learning workflow.
  4. Prepared GradCAM interpretation placeholders for visual model review.

Outputs

  • data/image_manifest.csv
  • outputs/model_metrics.csv
  • outputs/confusion_matrix.csv
  • figures/model_summary.svg

Interpretation

The demo metrics show the reporting structure expected for a real BBBC experiment: accuracy alone is not enough, so confusion patterns and image-level explanations are part of the workflow.
